5 AI Image Basics 기초 — 처음부터

Making an AI image is a conversation. You describe what you want in words; the AI draws it. If it's not right, you describe the change, and it draws again. That's the whole loop.

AI 이미지 만들기는 대화입니다. 원하는 것을 말로 설명하면 AI가 그립니다. 마음에 안 들면 바꿀 점을 설명하고 AI가 다시 그립니다. 이게 전부입니다.
  1. Open ChatGPT or Gemini on the PC. PC에서 ChatGPT나 Gemini를 엽니다.
  2. Type a description (a "prompt") of the image you want. 원하는 이미지의 설명("프롬프트")을 입력합니다.
  3. Look at what it made. 만들어진 것을 봅니다.
  4. Refine — tell it what to change ("simpler", "no text", "flat style"). 다듬기 — 바꿀 점을 말합니다.
  5. Download the image you like (right-click → Save). 마음에 드는 이미지를 저장합니다.

Repeat steps 3–4 as many times as you need. The first result is rarely the best — refining is normal, not failure. 3–4단계를 필요한 만큼 반복하세요. 첫 결과가 최고인 경우는 드뭅니다 — 다듬는 것은 실패가 아니라 정상입니다.

Two ways to work · 두 가지 길

(A) Redraw an existing image — what you'll do most today: upload a picture already in the workbook and ask the AI to redraw it in a fresh style (next section, §6). (B) Create from scratch — describe a brand-new image in words. Used less often. (A) 기존 이미지를 다시 그리기 — 오늘 대부분 이 방법: 학습서에 이미 있는 그림을 업로드해 새 스타일로 다시 그리게 함 (다음 §6). (B) 백지에서 만들기 — 새 이미지를 말로 설명. 덜 자주 씀.

6 Start from the Existing Image 기존 이미지에서 시작하기 — 오늘의 주된 방법

Most of the time you will not draw from a blank page. Instead you take an image that is already in the workbook and ask the AI to redraw it in a cleaner, modern style — keeping the same meaning. Work in three steps.

대부분은 백지에서 그리지 않습니다. 학습서에 이미 있는 그림을 가져와 AI에게 다시 그리게 합니다 — 같은 의미를 유지한 채 더 깔끔하고 현대적인 스타일로. 세 단계로 진행하세요.
  1. Understand it first. Ask: what is this image teaching? Read the workbook text around it. If unsure, give the AI the image and the nearby text and ask: "What is this diagram teaching? Explain each part." Miss the meaning and you get a pretty picture that is wrong. 먼저 이해하라. 이 그림은 무엇을 가르치는가? 주변 본문을 읽고, 확실치 않으면 AI에게 이미지 + 주변 본문을 주고 "이 도형이 무엇을 가르치나요? 각 부분을 설명해줘"라고 물으세요. 의미를 놓치면 예뻐도 틀린 그림이 됩니다.
  2. Upload and ask to redraw. Attach the image (📎 clip / image icon), then always tell it to keep the meaning, structure, and layout — change only the style: "Redraw this in a clean, modern, flat style. Keep the same meaning, layout, and all the parts — just clearer and more modern. No text inside (I'll add labels later)." 업로드하고 다시 그리게 하라. 이미지를 첨부(📎)한 뒤 항상 의미·구조·배치는 유지하고 스타일만 바꾸라고: "이 이미지를 깔끔하고 현대적인 플랫 스타일로 다시 그려줘. 같은 의미·배치·요소 모두 유지, 더 명확하게만. 안에 글자는 넣지 마."
  3. Compare with the original. Put them side by side. Same message? Anything missing, added, or changed by mistake? If so: "You removed the third arrow — put it back" / "keep it closer to the original." 원본과 비교하라. 나란히 놓고 — 같은 메시지인가? 빠지거나 더해지거나 바뀐 부분은? 있으면: "세 번째 화살표가 빠졌어 — 다시 넣어줘" / "원본에 더 가깝게."

Tips · 팁

  • Both ChatGPT and Gemini can edit/redraw an uploaded image. Try both. 둘 다 업로드 이미지를 편집·재생성 가능. 둘 다 시도.
  • If the original is complex, redraw one part at a time. 복잡하면 부분씩 다시 그리기.
  • Keep the original's text/labels and re-add them yourself — AI is weak at text. 원본 글자·라벨은 직접 다시 넣기 — AI는 글자에 약함.
  • The style words come from the prompt formula in §7. 스타일 표현은 §7 프롬프트 공식에서.

One simple rule for chats 채팅에 관한 단순한 규칙 하나

One image = one chat. Start a new chat for a new image. Keep understanding, drawing, and refining one image together in the same chat. When you move to a different image, open a fresh chat. If the AI keeps dragging in the old style or old parts, that's your signal to start a new chat.

한 그림 = 한 채팅. 새 그림은 새 채팅으로 시작. 하나의 그림을 이해·그리기·다듬기까지 같은 채팅에서 이어가고, 다른 그림으로 넘어가면 새 채팅을 여세요. AI가 자꾸 옛 스타일·요소를 섞으면 새 채팅을 열라는 신호입니다.

✎ Once you're comfortable — try these · 익숙해지면 시도해 보세요

  • If understanding got long, copy just the final prompt into a fresh chat and draw there — cleaner. 이해가 길어졌으면 최종 프롬프트만 복사해 새 채팅에서 그리기.
  • Try the same idea in both ChatGPT and Gemini and compare. 같은 아이디어를 둘 다에 넣고 비교.
  • Ask for 2–3 variations, then pick the best. 변형 2~3개 만들어 최고를 고르기.

You don't need to master all of this today. The coordinator is right here, and these habits become natural with practice. Start with the one rule above. 오늘 이걸 다 익힐 필요는 없습니다. 코디네이터가 바로 옆에 있고, 반복하면 자연스러워집니다. 위 규칙 하나부터 시작하세요.

7 How to Write a Good Prompt 좋은 프롬프트 쓰는 법

A good prompt answers a few simple questions. Use this formula:

Subject + Style + Composition + Color/Mood + Details + Format + What to avoid 대상 + 스타일 + 구도 + 색/분위기 + 세부사항 + 형식 + 피할 것
Part · 요소Example words · 예시 표현
Subject 대상"a shepherd guiding one sheep", "three steps of the gospel" 양을 인도하는 목자, 복음의 세 단계
Style 스타일flat illustration, line drawing, minimal, warm watercolor 플랫 일러스트, 선화, 미니멀, 수채화
Composition 구도centered, left-to-right flow, top-down 중앙, 왼쪽→오른쪽, 위→아래
Color/Mood 색·분위기soft, hopeful, 2–3 colors only 부드럽고 소망 있는, 색 2~3개만
Details 세부simple faces, no crowd 단순한 얼굴, 군중 없이
Format 형식square, white background, space for a label 정사각형, 흰 배경, 라벨 자리
Avoid 피할 것no text, no clutter, not too realistic 글자 없이, 복잡하지 않게

WEAK · 약함   "a picture about faith"

STRONG · 강함   "A simple flat illustration of a single open hand receiving a small light, warm colors, white background, minimal, no text, space at the bottom for a caption."

강한 프롬프트: "한 손이 작은 빛을 받는 단순한 플랫 일러스트, 따뜻한 색, 흰 배경, 미니멀, 글자 없이, 아래쪽에 설명을 넣을 여백."

Refining words that work well: "simpler" · "flat / minimal" · "fewer colors" · "remove the text" · "center it" · "same style as before" · "make the background white / transparent". 잘 통하는 다듬기 표현: "더 단순하게" · "미니멀하게" · "색 줄여서" · "글자 빼고" · "중앙 정렬" · "이전과 같은 스타일로" · "배경 흰색/투명으로".

8 Making Each Image Type Well 각 유형별로 잘 만들기

8a. Concept Diagrams 개념 도형

These show relationships: steps, arrows, boxes. AI is not good at putting exact text inside diagrams — letters often come out misspelled. So keep text minimal or none, ask for clear shapes and arrows, and add the labels yourself afterward. ChatGPT usually handles text better if you must include a word.

관계를 보여주는 그림입니다. AI는 도형 안 정확한 글자에 약합니다 — 글자가 자주 틀립니다. 글자는 최소화하거나 없이, 명확한 도형·화살표를 요청하고, 라벨은 나중에 직접 넣으세요. 예: "화살표로 연결된 상자 세 개를 왼쪽에서 오른쪽으로, 플랫 스타일, 안에 글자 없이."

8b. Illustrations / Scenes 삽화·장면

  • One style per workbook. Pick a style and keep every image in that book the same. Reuse "same style as before." 학습서 하나당 스타일 하나. "이전과 같은 스타일로" 재사용.
  • Be faithful and reverent. Match the Bible. Keep people modest and culturally neutral. Do not depict the face of God — suggest the divine with light. 충실·경건하게. 인물은 단정·중립적. 하나님 얼굴은 그리지 말고 빛으로 암시.
  • Keep it universal. Avoid clothing or settings tied to one single culture — these workbooks travel to many nations. 특정 한 문화에만 얽매인 복장·배경은 피하세요 — 여러 나라로 갑니다.

8c. Icons / Symbols 아이콘·심볼

Small, simple marks — a cross, a book, a lamp. Ask for: "a single simple flat icon of a cross, minimal line style, one color, white/transparent background." Make a matching set — same line weight and color — so the workbook feels unified.

작고 단순한 표시. 요청 예: "십자가 하나의 단순한 플랫 아이콘, 미니멀 선 스타일, 한 색, 흰/투명 배경." 선 굵기·색을 통일한 세트로 만드세요.

9 Quality Checklist 품질 점검 — 확정 전에

Ask these before saving. If any answer is "no," refine or try again.

  • Same message? If you redrew an existing image, does it still teach the same thing? 같은 메시지인가? 다시 그렸다면 여전히 같은 것을 가르치는가?
  • Clear? Understandable in 2 seconds? 명확한가? 2초 안에 이해되는가?
  • Serves the lesson? Helps the point, doesn't distract? 배움을 돕는가? 방해하지 않는가?
  • Consistent? Same style as the other images here? 일관적인가? 다른 그림과 스타일이 같은가?
  • Text legible? Words spelled right? (If not, remove & add later.) 글자가 정확한가? (아니면 지우고 나중에 추가)
  • Appropriate & reverent? Modest, universal, God not a figure? 적절·경건한가? 단정·보편적, 하나님을 인물로 안 그렸는가?
  • Faithful? True to the Bible and the workbook's meaning? 충실한가? 성경·학습서 의미에 맞는가?
  • Good resolution? Sharp, large enough to print? 해상도가 좋은가? 인쇄할 만큼 큰가?
  • Simple? Could a local believer understand and reuse it? 단순한가? 현지 신자도 이해·재사용 가능한가?
